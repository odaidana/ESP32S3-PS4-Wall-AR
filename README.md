# ESP32S3-PS4-Wall

أداة مستقلة لحجب الشبكة وتوجيه الـ DNS تعمل على شريحة ESP32-S3 (N16R8). يعمل المشروع كجدار حمايه محلي لجهاز الـ PS4 لمنع التحديثات الإجبارية عبر فصل الجهاز عن الإنترنت الخارجي بالكامل وتوجيهه لصفحة محلية آمنة.

## الهدف من المشروع
ربط أجهزة الـ PS4 المعدلة أو التي تمتلك نظام قديم بالإنترنت يشكل خطراً كبيراً بسبب تحديثات سوني التلقائية. هذا المشروع يقوم بإنشاء نقطة اتصال واي فاي معزولة تماماً، حيث يتم إسقاط وحجب أي اتصال إنترنت خارجي لحماية الجهاز، مع توجيه أي محاولة تصفح أو فتح دليل المستخدم إلى صفحة محلية تؤكد نجاح الحجب التام.

## المكونات المطلوبة
- لوحة تطوير ESP32-S3 (نسخة N16R8).
- كابل USB-C يدعم نقل البيانات.

## آلية العمل
1. **نقطة اتصال معزولة:** تبث البوردة شبكة واي فاي مغلقة وخاصة بها.
2. **حجب كامل للإنترنت:** لا تقوم البوردة بتمرير أي بيانات للإنترنت الخارجي، مما يحافظ على الكونسول أوفلاين بنسبة 100%.
3. **توجيه الـ DNS التلقائي:** يتم اعتراض أي طلب تصفح من الـ PS4 وعرض صفحة محلية مخصصة مكتوب فيها: `REDIRECT WEB FOR ADVANCED NETNAME(NETNAME)`.

## طريقة التركيب والتشغيل
1. افتح كود المشروع باستخدام بيئة **Arduino IDE**.
2. اختر نوع البوردة ESP32-S3 من مدير اللوحات.
3. ارفع كود الـ C++ المرفق في المستودع.
4. من جهاز الـ PS4، ابحث عن شبكات الواي فاي واشبك على شبكة `PS4-Secure-Wall`.
